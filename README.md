# PineappleChat

A simple real-time anonymous chat app for talking to strangers. Built with Node.js + Socket.IO.

## Features

- Completely anonymous (no accounts)
- 18+ age gate and safety agreement
- Explicit choice from online users list
- Multi-chat, filters, notifications
- No links / unsafe language in chat
- Light & dark modes
- Soothing pineapple-inspired theme

## Run it

```bash
npm install
npm start
```

Open **http://localhost:3000** in your browser.

**Pro tip:** Open 2–3 tabs to test chatting with yourself.

## Tech

- Express + Socket.IO (server)
- Vanilla HTML/JS + Tailwind CDN (client)
